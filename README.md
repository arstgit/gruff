# gruff

[![Build Status](https://travis-ci.org/derekchuank/gruff.svg?branch=master)](https://travis-ci.org/derekchuank/gruff)

Convert a bmp image to a ascii image. `Linux only`.

## Compile & Install

```
  $ make && make install
```

## Usage

```
  $ gruff example.bmp
```
